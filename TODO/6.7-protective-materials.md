# §6.7 — Protective Materials (G1)

> Corresponds to PRD §6.7 Protective Materials (FR-PM-01 – FR-PM-14).
> Note: FR-PM-* here refers to the Protective Materials requirements namespace,
> distinct from the Project Management FR-PM-* namespace in §6.1.

## Material Library (§6.7.1)

- [ ] Add `protective_material` component type with categories: `felt_tape`, `fabric_tape`, `heat_shrink`, `corrugated_loom`, `smooth_loom`, `spiral_wrap`, `conduit`, `rubber_grommet`, `cable_tie`, `other`
- [ ] Category-specific properties (width, OD, shrink ratio, split flag, adhesive side, temperature rating, etc.)
- [ ] Ship built-in catalog entries (Hellermann Tyton, TE, Raychem part numbers) (FR-PM-10)
- [ ] Protective material components support image and STEP file attachments (FR-PM-11)

## Applying Materials (§6.7.2)

- [ ] Coverage span model: (target wire/cable/bundle/fan-out, start_offset_mm, end_offset_mm) (FR-PM-01)
- [ ] Reference full length, named segment, or arbitrary offset from End A (FR-PM-02)
- [ ] Multiple materials on same span with `layer_order` for nesting (FR-PM-03)
- [ ] Visualize coverage spans on canvas as colored overlay bands with legend; stack overlapping layers (FR-PM-04)
- [ ] Add coverage span via Properties Panel **Add protection** button (FR-PM-05)
- [ ] Add coverage span via right-click → **Add protection here** (pre-fill offset to click point) (FR-PM-05)
- [ ] Coverage span editor fields: material, start/end offset, named segment, layer order, quantity/wrap coverage, notes (FR-PM-06)
- [ ] Calculate total tape length: span length × tape width × overlap % (FR-PM-07)
- [ ] Check loom/conduit inner diameter ≥ bundle outer diameter of covered conductors (FR-PM-08)
- [ ] Check heat-shrink pre-shrink inner diameter ≥ conductor/cable OD; display post-shrink diameter informatively (FR-PM-09)

## DRC Rules (§6.7.3)

- [ ] PROT-01 Warning: loom/conduit inner diameter < bundle OD at any covered cross-section
- [ ] PROT-02 Warning: heat-shrink pre-shrink inner diameter < conductor/cable OD
- [ ] PROT-03 Info: coverage span extends beyond the wire/cable's defined length
- [ ] PROT-04 Warning: overlapping spans of same material type and layer order
- [ ] PROT-05 Warning: cable tie spacing > 300 mm on unsupported span (configurable threshold)

## In Documents (§6.7.4)

- [ ] BOM: protective materials as their own section; tape/wrap in metres, loom/conduit in metres, grommets/ties in units (FR-PM-12)
- [ ] Cut sheet: "Protection" table per wire/cable listing coverage spans in End A order: material part number, start–end offsets, layer, calculated quantity (FR-PM-13)
- [ ] Assembly Drawing document: schematic-level PDF with coverage bands as colored overlays, material legend, callout labels (FR-PM-14)
