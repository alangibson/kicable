# §6.2 — Schematic Editor (G1)

> Corresponds to PRD §6.2 Schematic Editor (FR-SE-01 – FR-SE-10).

- [x] Infinite, pannable, zoomable canvas using React Flow (FR-SE-01)
- [x] Drag connectors from library panel onto canvas to place nodes (FR-SE-02)
- [x] Draw wires by clicking a pin and dragging to another pin; snap to valid pins only (FR-SE-03)
- [x] Add waypoints to wire segments by click-drag on the segment (FR-SE-04)
- [x] Insert splice nodes on any wire (3-way and 4-way junctions) (FR-SE-05)
- [x] Select any node or wire; show properties in right-hand Properties Panel (FR-SE-06)
- [x] Multi-select via rubber-band drag; support group move (FR-SE-07)
- [x] Undo/redo stack with ≥ 100 steps (FR-SE-08)
- [x] Auto-save on every user action, debounced at 2 s, targeting `StorageAdapter` (FR-SE-09)
  - [x] Run auto-save in a background worker to avoid blocking the UI thread (NFR-R-01)
- [x] Keyboard shortcuts: delete, undo, redo, zoom in/out, pan (FR-SE-10)
- [x] Canvas renders 500+ nodes and 1000+ wires at ≥ 60 fps (NFR-P-01)
