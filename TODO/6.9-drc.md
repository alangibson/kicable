# §6.9 — Validation & Design Rule Checks (G1)

> Corresponds to PRD §6.9 Validation & Design Rule Checks (FR-DV-01 – FR-DV-07).
> Rule implementations live in their respective feature files; this file covers the
> DRC engine, runner, and results UI.

- [ ] DRC runs automatically on every save and on-demand via toolbar button (FR-DV-01)
- [ ] Schematic rules (FR-DV-02):
  - [ ] Unconnected pins marked as required
  - [ ] Duplicate wire IDs in the same project
  - [ ] Wire gauge mismatch between connected pins
  - [ ] Circular routes (wire connects a pin to itself)
  - [ ] Missing signal names on wires flagged as "named required"
- [ ] Length and strip rules — see `6.4-wire-dimensions.md` (FR-DV-03)
- [ ] Cable split/join rules — see `6.6-cable-split-join.md` (FR-DV-04)
- [ ] Protective material rules — see `6.7-protective-materials.md` (FR-DV-05)
- [ ] DRC results panel: grouped by category (Schematic, Lengths, Split/Join, Protection); clickable items zoom canvas to offending element (FR-DV-06)
- [ ] Errors block document export; warnings and info messages are non-blocking (FR-DV-07)
