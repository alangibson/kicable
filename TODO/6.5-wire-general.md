# §6.5 — Wire & Cable General Properties (G1)

> Corresponds to PRD §6.5 Wire & Cable General Properties (FR-WG-01 – FR-WG-05).

- [x] Wire properties: wire ID (auto-generated, editable), gauge (AWG or mm²), color, signal name, notes (FR-WG-01)
- [x] Wire color standards: ISO 6722, SAE J1128, or custom; show color swatch on canvas edge (FR-WG-02)
- [x] Group wires into bundles; auto-calculate bundle outer diameter from gauges and fill ratio (FR-WG-03)
- [x] Auto-propagate signal names to matching pins across schematic (FR-WG-04)
- [x] Place cables as single routed entities; inner conductors inherit routed path but have individual signal names, colors, and strip defs (FR-WG-05)
